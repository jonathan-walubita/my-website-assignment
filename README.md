 My Website Assignment (IWD)

Description
This is a personal HTML5 website created for my Introduction to Web Development (IWD) coursework. It demonstrates the use of semantic HTML elements, attributes, and basic web structure.

Objective
- To understand how HTML5 works
- To build a structured personal website
- To practice semantic HTML elements
- To learn basic web development workflow

Website Content
- Header section
- Introduction paragraph
- Skills list
- Article section
- Footer section
- External link (GitHub)

HTML Elements Used
- header
- section
- article
- footer
- ul and li
- a (anchor tag)

Challenges Faced
The most challenging elements were:
- section: understanding how to divide content properly
- article: learning how self-contained content works
- semantic tags: deciding where each section belongs
- ul/li: structuring lists correctly
- linking (a tag): managing external links properly

Semantic HTML Usage
Semantic elements like header, section, article, and footer were used to structure the webpage in a meaningful and readable way. This improves clarity and organization.

Most Useful Element
The section element was the most useful because it helped divide the webpage into clear parts, making the structure easy to understand.
