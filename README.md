 Question 2: HTML Elements
1. Most Challenging Elements
The five most challenging elements were:
<section> – understanding how to divide content logically
<article> – knowing when to use it as independent content
<header> – structuring the top part correctly
<footer> – placing correct information at the bottom
<a> – handling links properly
These were challenging because they required understanding of proper page structure.
2. Semantic HTML Usage
Semantic elements like <header>, <section>, <article>, and <footer> were used to organize the webpage into meaningful parts. This improves readability and makes the website easier to understand.
3. Most Useful Element
The <section> element was the most useful because it helped divide the webpage into clear sections such as introduction, skills, and content areas.
 Question 3: HTML Attributes
1. Essential Attributes
The three most important attributes used were:
href – used to create links
src – used to display images
alt – used to describe images
These are essential for functionality and accessibility.
2. class vs id
id is used for unique elements (only one per page)
class is used for grouping multiple elements
This helps in organizing and identifying elements.
3. Attribute that improved user experience
The href attribute improved user experience the most because it allows users to navigate to other pages or websites easily.
